┌──────────────────────────────────────────────────────────────────────────────┐
│  ✨ HOW IT WORKS                                                             │
└──────────────────────────────────────────────────────────────────────────────┘

    1. You drop MP4 files in church_videos folder
                    ↓
    2. Video server scans the folder automatically
                    ↓
    3. Server generates thumbnails from first frame
                    ↓
    4. Server creates JSON list of all videos
                    ↓
    5. Reels page fetches the list via API
                    ↓
    6. Videos appear in beautiful Instagram-style gallery!

    Problem: "Video Server Not Running"
    Solution: Run START_VIDEO_SERVER.bat or node get_videos.js

    Problem: "No Videos Found"
    Solution: Add MP4 files to church_videos folder

    Problem: Thumbnails not showing
    Solution: Wait for auto-generation or add manual JPG files

    Problem: Videos won't play
    Solution: Check video format (must be MP4)
