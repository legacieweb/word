┃  🎮 HOW TO USE THE REELS PLAYER                                            ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

    GALLERY VIEW:
    ─────────────
    🖱️ Click any video thumbnail → Enter watch mode
    👁️ See all your videos in a beautiful grid

    WATCH MODE:
    ───────────
    📱 Swipe up/down (or scroll) → Next/Previous video
    ❤️ Heart button → Like the video
    ↗️ Share button → Share with others
    🔇/🔊 Speaker button → Mute/Unmute sound
    ✕ Close button → Back to gallery

    KEYBOARD SHORTCUTS:
    ───────────────────
    ⬆️ Arrow Up → Previous video
    ⬇️ Arrow Down → Next video
    L → Like current video
    Esc → Close watch mode

    ✅ Solution: Run START_VIDEO_SERVER.bat or node get_videos.js

    ❌ Problem: "No Videos Found"
    ✅ Solution: Add MP4 files to church_videos folder

    ❌ Problem: Node.js not installed
    ✅ Solution: Download from https://nodejs.org/

    ❌ Problem: Videos won't play
    ✅ Solution: Make sure files are .mp4 format
