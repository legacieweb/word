🚀 HOW TO START A SERMON (NEW SIMPLIFIED PROCESS)
═══════════════════════════════════════════════════════════════════════════

  1. Open your browser and go to:
     
     👉 http://localhost:3000/admin.html
     
     ⚠️ IMPORTANT: Use the HTTP address above!
        DO NOT double-click the HTML file!

  2. Click "Go Live" in the sidebar

  3. Enter your sermon details:
     • Sermon Title (e.g., "Sunday Morning Service")
     • Admin PIN (your secret PIN)

  4. Click "Start Live Sermon" button

  5. That's it! You're now live! 🎉


📊 WHAT YOU'LL SEE WHILE LIVE
═══════════════════════════════════════════════════════════════════════════

  • ⏱️ Live Duration - How long you've been broadcasting
  • 👥 Viewer Count - Number of people watching
  • 💬 Message Count - Chat messages from viewers
  • 💾 Recording Size - Size of your audio recording


🛑 HOW TO STOP THE SERMON
═══════════════════════════════════════════════════════════════════════════

  1. Click the "Stop Live Sermon" button
  
  2. Confirm you want to stop
  
  3. Your recording will be automatically saved! ✅


🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

  ❌ Problem: Can't access admin panel
  ✅ Solution: Make sure server is running
              Run: node D:\desktop\church\server.js

  ❌ Problem: CORS errors in console
  ✅ Solution: Use http://localhost:3000/admin.html
              NOT file:///D:/desktop/church/admin.html

  ❌ Problem: Microphone not working
  ✅ Solution: Allow microphone permission in browser
              Check browser settings
