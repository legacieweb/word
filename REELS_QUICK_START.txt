🚀 HOW TO USE (3 STEPS):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Add Videos
──────────────────
Copy your MP4 files to:
📁 d:\desktop\church\church_videos\

Example:
  copy "C:\Videos\Sunday_Worship.mp4" "d:\desktop\church\church_videos\"


STEP 2: Start Server
─────────────────────
Open Command Prompt and run:
  cd d:\desktop\church
  node server.js

You should see:
  ✅ Connected to MongoDB
  🚀 Server is running at http://localhost:3000
  📹 Video API available at http://localhost:3000/api/videos


STEP 3: Open Reels
───────────────────
Open in your browser:
  http://localhost:3000/reels.html

Click any video → It plays with SOUND! 🔊
