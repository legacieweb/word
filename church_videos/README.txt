HOW TO ADD VIDEOS:
==================

1. Copy your MP4 video files into this folder
2. Name them descriptively (e.g., "Sunday_Worship.mp4")
3. Start the video server (run START_VIDEO_SERVER.bat)
4. Open reels.html in your browser
5. Your videos will appear automatically!
